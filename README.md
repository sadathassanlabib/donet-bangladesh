# 📌 Donation App (Famine Relief – Feni 2024)

A simple donation web application built using **Vanilla JavaScript** and **Tailwind CSS**.  
This app allows users to donate money from their balance, updates the UI instantly, and keeps a donation history log with date and time.

---

## 🚀 Features

- 💰 Donate money from available balance
- 🧮 Real-time balance update
- 🗂️ Donation history with timestamp
- ✅ Proper input validation
- 🎨 Clean and responsive UI with Tailwind CSS
- ⚡ Lightweight and fast (No frameworks)

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript (DOM Manipulation)

---

## 🧠 Core JavaScript Concepts Applied

- DOM selection and manipulation
- Event handling
- Form validation
- Dynamic element creation
- Reusable functions
- Clean code structure

---

## ⚙️ How It Works

1. User enters a donation amount.
2. The system validates the input.
3. The donation amount is added to the card.
4. The user balance is reduced instantly.
5. A history entry is created with date and time.

---

## ❗ Validation Rules

- Donation cannot be empty or invalid
- Donation cannot be zero or negative
- Donation cannot exceed available balance
